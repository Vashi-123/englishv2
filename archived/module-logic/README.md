# Архив логики модулей

Эта папка содержит логику модулей (vocab, grammar, correction, practice), которая была удалена из основного приложения, но сохранена для возможного будущего использования.

## Содержимое

- `moduleLogic.ts` - Определения модулей и вспомогательные функции для работы с модулями

## Примечание

Логика модулей была удалена из приложения, так как больше не используется. Весь функционал теперь работает без разделения на модули.

Если в будущем понадобится вернуть логику модулей, можно использовать код из этой папки как основу.
